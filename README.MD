
# Compile and Run C File

## Using Command Prompt

**Compile:**
```cmd
gcc filename.c -o filename.exe
```

**Run:**
```cmd
filename.exe
```

**Compile and Run (combined):**
```cmd
gcc filename.c -o filename.exe && filename.exe
```

## Alternative (using MinGW or MSVC)

**With MSVC:**
```cmd
cl filename.c
filename.exe
```

## Notes
- Replace `filename.c` with your actual C file name
- Ensure `gcc` is installed and added to PATH
- Use `./filename.exe` on some systems
